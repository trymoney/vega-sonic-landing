# Vega-Sonic Command Center

Lead Generation Dashboard for Vega-Sonic VS-12 Ultrasonic Gas Flow Meter

## Quick Start

### Windows
```bash
install.bat
npm start
```

### Linux/Mac
```bash
chmod +x install.sh
./install.sh
npm start
```

Then open: **http://localhost:3000**

## Features

- Landing Page with product info
- Dashboard with leads management
- Multi-language support (EN, ES, RU)
- Responsive design

## Requirements

- Node.js 18+
- npm 9+

## Tech Stack

- React 18
- Vite
- i18next
- Express.js
